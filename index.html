Usage

  • Create a div with a class of your choice
    For the purpose of this example, I took it to be 'grid'.
  • Create a collection of images and save them in a sequential order
    I stored them in a 'img' folder with names from
    1 to 21 and extention '.jpg'.
  • Decide a width and height of the images
    I chose width = 125px and height = 75px
  • Chose the size of buttons
    I chose this to be 30px
  • Finally chose number of rows and columns
    I chose rows = 4 and columns = 4
